# description

Opens a list of PRs that "need your review". The selected pr will open in the browser.

# installation
make sure you have the dependencies
```shell
brew install gh
brew install fzf
```
install the extension
```shell
gh extension install axkirillov/review
```

# usage

```shell
gh review
```
